Projects

Here you will find a list of web projects made by me.

Homepage

This page contains a collection of fun, popular and useful websites organised into sections. It is very customisable, you can add your own collection of websites in a separate section. This project was inspired by the start screen from windows 8.

IRC Quote Prettifier

This tool automatically prettifies IRC quotes. It has options to change the message syntax, hide IPs, HTML colour nicknames, convert IRC formatted text to HTML, HTML encode special characters and preview the result. It is useful for posting quotes in forums.

IRC Tutorial

This page has 3 web clients you can choose from. It shows you information about the web client, how to connect, useful things to know about, commands and how to behave. It also allows you to use colours and other text formats for web clients.

Population Pyramid Generator

This tool generates a semi-accurate population pyramid using 4 variables entered by the user, crude birth rate, crude death rate, sex ratio and net migration rate.
