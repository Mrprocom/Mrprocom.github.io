Quick Access List Info

Here you can add pages you would like to be added to the quick access list, which can be accessed from the navigation bar up top.
Syntax:

<Link> <Title>

Example:

https://google.com/ Google Search Engine
https://youtube.com/ YouTube
https://mail.google.com/ Gmail

Custom Pages Info

Here you can add your favourite pages to be added to the Custom Pages section, which can be accessed from clicking the Custom tile from the main section.

  • Do not use spaces in section names, use underscores _ Underscores automatically get converted to spaces when displayed.
  • Each section contains 4 columns, a wide tile expands to a full column and a small tile only expands to half a column, so 2 small tiles are needed to fill a column.
  • There is no limit to the number of sections.
  • Section names are case-sensitive, the main section is called Main with an upper-case M.
  • There are 2 tile types: section (redirects to a section) and website (redirects to another website).
  • There are 2 tile sizes: small and wide.
  • There are 5 colours: blue, green, orange, purple and red.
  • There are 2 title types: icon (image with label) and text (text in the middle of the tile).
  • If you want to use existing icons, use icons/icon-name.svg as the icon link. List of existing icons.
  • A good website to find icons is Flaticon, most icons used here were taken from there.
  • You can learn from the default Custom Pages to make your own.
  • It is recommended to backup your custom pages text in a local text file on your device in case you accidentally lose it here.

Section Tile Syntax:

<Section Name> <Column> section <Direct-to Section Name> <Tile Size> <Tile Colour> text
<Section Name> <Column> section <Direct-to Section Name> <Tile Size> <Tile Colour> icon <Icon Link>

Website Tile Syntaxes:

<Section Name> <Column> website <Page Link> <Tile Size> <Tile Colour> icon <Icon Link> <Label>
<Section Name> <Column> website <Page Link> <Tile Size> <Tile Colour> text <Title>
