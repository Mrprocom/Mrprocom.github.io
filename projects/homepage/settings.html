Quick Access List Info

Here you can add pages you would like to be added to the quick access list, which can be accessed from the navigation bar up top.
Syntax:

<Link> <Title>

Example:

https://google.com/ Google Search Engine
https://youtube.com/ YouTube
https://mail.google.com/ Gmail

Custom Pages Info

Here you can add your favourite pages to be added to the Custom Pages section, which can be accessed from clicking the Custom tile from the main section.

  • The custom page is separated into 4 columns, each column with its own tiles.
  • There are 2 tile types: section (redirects to an existing section) and website (redirects to another website).
  • There are 15 sections: access, communication, entertainment, games, knowledge, media, microsoft, miscellaneous, news, notes, search, social, store and tools
  • There are 2 tile sizes: small and wide.
  • There are 5 colours: blue, green, orange, purple and red.
  • There are 2 title types: icon (image with label) and text (text in the middle of the tile).
  • If you want to use existing icons, use icons/icon-name.svg as the icon link. List of existing icons.

Section Tile Syntax:

section <Section Name> <Tile Size> <Tile Colour> <Title Type>

Example:

section search wide red icon
section social small blue text
section news small purple text

Website Tile Syntaxes:

website <Page Link> <Tile Size> <Tile Colour> icon <Icon Link> <Label>
website <Page Link> <Tile Size> <Tile Colour> text <Title>

Example:

website https://google.com/ wide blue text Google Search Engine
website https://youtube.com/ small red icon http://example.com/img.png YouTube
website https://mail.google.com/ small red icon icons/gmail.svg Gmail
