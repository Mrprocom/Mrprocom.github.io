IRC Tutorial

Introduction

IRC stands for Internet Relay Chat. You can think of it as a chatting system where you have so many chatting rooms to choose from.

An IRC client is a program that lets users connect to IRC. There are some IRC web clients like QwebIRC and KiwiIRC as well as some clients that you can install like HexChat, mIRC and irssi.

There are many IRC servers that you can connect to like Freenode, which is the most popular one and the one you should always connect to. Each server has its own IRC channels (some servers have the same channels).

In this tutorial, you will learn the basics of using the IRC. You will be using QwebIRC for now and if you feel like you want to switch to KiwiIRC or install an IRC client, you can do that.

How to Connect to the Server

  • Type in the nickname that you want to be known as in the Nickname text field. Your nickname should represent yourself so that users can recognise you.
  • Type in the channel names that you want to join after connecting to the server in the Channels text field. This is optional, meaning that you can leave it empty if you do not want to join any channel after connecting. If you want to join multiple channels at once, just separate the channel names with a comma. Channels have to start with at least 1 hashtag.
  • If you do not have a NickServ account, then skip this step, otherwise check the "Auth to services" checkbox and type in your username and your password.
  • The reCAPTCHA is only used to make sure that the user is a human and not a robot. This is useful to prevent spam and abuse. Just do what the it says, it will only take a few seconds.
  • Click on the "Connect" button to connect to the server and if you see an error message, you might want to fix something and try again.

Simple Concepts

Take a look at this image, it names and describes some of the parts of the client.
Here are some useful things that you should know about:

  • If you want to get someone's attention, just mention their nickname, they will receive a notification and be able to read your message.
  • Do not add a "+" or an "@" at the beginning of the operator or the voiced user that you want to highlight.
  • The best way to mention someone's nickname is by typing the first few letters of their nickname and pressing the tab key to auto-complete it.
  • Not all channels have their moderators operated and the trusted users voiced.
  • Channel names and nicknames are case-insensitive.

Useful Commands

Commands start with a "/" and are case-insensitive. You do not have to memorise all of them, but you should know these:

Syntax Description
/AWAY <Reason> Marks you as being away (better than disconnecting)
/BACK Marks you as being back (the opposite of /AWAY)
/HELP [<Command>] Gives you a list of commands or information about the command you gave it
/JOIN <Channel> Joins that channel
/ME <Action> Sends the action in third person
/MSG <Nick> <Message> Sends a private message to that user
/NICK <Nick> Changes your nickname
/PART [<Channel>] [<Reason>] Leaves that channel (the current one by default)
/QUERY <Nick> [<Message>] Opens up a new private message window to that user and optionally sends a message
/QUIT [<Reason>] Quits from the server

Tips and Advice

Congratulations! you have reached this section which is the most important one.
This is a list of things that you should follow and keep in mind:

  • Follow the rules of the channel that you are visiting (read the topic of the channel).
  • Do not spam or flood (including rejoining the same channel and changing the nickname rapidly). If you want to, you can open up a new query with yourself and spam there or spam in a random and abandoned channel that does not have any users.
  • Do not highlight users for no reason or mass-highlight the channel.
  • Do not troll.
  • Do not ban evade.
  • Do not abuse caps lock or formatted characters.
  • Do not be annoying.
  • Do not be very rude to people, be friendly to them and to new users instead.
  • Do not impersonate other users.
  • Do not ask for channel powers if you do not have rights to ask them.
  • Do not worry if users are not talking, they just talk whenever they have anything to say.

These rules should be strictly followed depending on how important, popular and big the channel is. Have fun!
