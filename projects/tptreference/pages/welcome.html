Welcome

Welcome to the Ultimate Powder Toy Reference. This reference includes a lot of useful things about the game. Read the following instructions for a better browsing experience.

  • Click on a topic title from the column on the left to read about it.
  • It is recommended to view them in order for beginners.
  • Clicking a topic or a subtopic within a topic will change the URL of the page. The new URL represents a direct link to that specific topic/subtopic, which can be shared to anyone if desired.
  • Holding Shift while clicking any link changes this area of the page to view the page the link redirects to. This is useful for opening links without leaving the page. Not all links can be viewed this way.
  • Holding Ctrl+Shift while clicking any link opens up the link as a dialog box, this also allows viewing links without leaving the current page. Not all links can be viewed this way.
  • Click the Quick Search button on this page to quickly navigate to a specific topic or subtopic of the reference. Enter keywords in the search bar that exist in the title or the ID of the topic/subtopic you want to navigate to.
  • Press E while being on any topic to bring up the Quick Search dialog box with a search query that shows all subtopics of the current topic. This helps navigate quickly within the current topic.
