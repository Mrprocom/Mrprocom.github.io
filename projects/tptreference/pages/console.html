Console

The console is a window you can bring up where you can run built-in commands or Lua commands. You can access it by pressing the C quick option or by pressing the ~ key. All built-in commands start with an exclamation mark !, if it does not, then it will be considered a Lua command. Pressing the up arrow key while in the console inputs the last command you ran, pressing it more inputs the command before it and so own. Pressing the down arrow key does the opposite, if you press the up arrow key multiple times, you can press the down arrow key to input the more recent commands.

The Set Command

This command changes the value of any property of particles.

Syntax: 
!set <Property> <Element> <Value>
Explanation:
  • Property: The property you want to change its value: type, ctype, life, temp, tmp, tmp2, flags, dcolor, pavg0, pavg1, x, y, vx, vy.
  • Element: The short element name (metl or METL for example) or all to target all particles in the simulation or the particle index.
  • Value: The value you want to set the property of the targeted particles to. This could be a number or an element name or none sometimes.
Examples:
  • !set type dust metl: Turns all DUST into METL.
  • !set type dust none: Removes all DUST.
  • !set type 3 metl: Turns the particle with an index of #3 to METL.
  • !set ctype clne 3: Changes the Ctype value of all CLNE to the element with an ID of 3, which is OIL.
  • !set ctype clne watr: Changes the Ctype value of all CLNE to WATR.
  • !set temp all 1000: Changes the temperature of all particles to 1000 Kelvin.
  • !set temp all 1000C: Changes the temperature of all particles to 1000 Celsius.
  • !set temp all 1000F: Changes the temperature of all particles to 1000 Fahrenheit.
  • !set life phot 1000: Changes the Life value of all PHOT to 1000.
  • !set life stkm 1000: Changes the Life value of all STKM to 1000.
  • !set dcolor metl 0xffffffff: Changes the colour of all METL to 0xFFFFFFFF, which is the colour white.

The Create Command

This command creates a particle in specified coordinates.

Syntax: 
!create <Element> <Coordinates>
Explanation:
  • Element: The short element name (metl or METL for example) or the element ID.
  • Coordinates: The X and Y coordinates of where the element will be created. The X and Y values are separated by a comma.
Examples:
  • !create dust 300,200: Creates one pixel of DUST in X:300, Y:200.
  • !create 3 300,200: Creates one pixel of the element with an ID of 3, which is OIL, in X:300, Y:200.

The Delete or Kill Command

This command deletes a particle in specified coordinates or deletes a particle with a specified index.

Syntax: 
!delete <Coordinates or Index>
Explanation:
  • Coordinates or Index: The X and Y coordinates of the particle that will be deleted or the index of the particle that will be deleted. The X and Y values are separated by a comma.
Examples:
  • !delete 300,200: Deletes the particle in X:300, Y:200.
  • !kill 300,200: Deletes the particle in X:300, Y:200. There is no difference between !delete and !kill.
  • !delete 3: Deletes the particle with an index of #3.

The Reset Command

This command can resets velocities, pressures, temperatures and sparks.

Syntax: 
!reset <What to Reset>
Explanation:
  • What to Reset: The thing you want to reset. This could either be velocity, temperature, pressure or sparks.
Examples:
  • !reset velocity: Resets velocities.
  • !reset temp: Resets all temperatures back to their original temperature when they were drawn.
  • !reset pressure: Resets all pressure back to 0.
  • !reset sparks: Removes all SPRK leaving the conductor behind.

The Bubble Command

This command creates a big SOAP bubble in the coordinates.

Syntax: 
!bubble <Coordinates>
Explanation:
  • Coordinates: The X and Y coordinates of where the bubble will be created. The X and Y values are separated by a comma.
Example:
  • !bubble 300,200: Creates a SOAP bubble in X:300, Y:200.

The Load Command

This command loads a save with a specified ID.

Syntax: 
!load <Save ID>
Explanation:
  • Save ID: The ID of the save that will be loaded.
Example:
  • !load 2198: Loads the save with the ID 2198, which is the Desert City 5 save.

The Quit Command

This command closes the game instantly without asking for confirmation.

Syntax: 
!quit
