Notable Users

The difference between moderators and developers is that moderators moderate the forum and in-game saves and they also deal with users who violate the rules. Developers develop the game, they modify the source code to improve it, fix bugs or add new features. Some users can be both moderators and developers at the same time, but it cannot be clearly seen on their profile.

If there is something important you want to discuss with a moderator, start a conversation with any of them by clicking the Send Message button on their profile. If it is just about reporting a save for breaking any rules, you should report that save instead, that way, it will be seen by any moderator, so it will receive quicker attention than sending a conversation to a moderator reporting about a save.

Special Roles

  • Simon: The administrator and one of the original creators of the game. He owns the Powder Toy server and has full control over it.
  • Skylark: The original creator of the game. You can find him on IRC.
  • Xenocide: The original owner of an IRC bot called StewieGriffin, which later got replaced with PowderBot; the current Powder Toy IRC channels bot owned by jacksonmj.
  • jacob1: A moderator and developer who has access to the website's code and can modify it.
  • jacksonmj: Also a moderator and developer who has access to the website's code. He is also the owner of PowderBot, which is the official bot for the Powder Toy channels on IRC, and the #powder Stats.
  • jacobot: A bot owned by jacob1. It is used to make moderation easier via IRC. Although it is not labelled as a moderator on the website, its username is highlighted in blue in the game just like any other moderator.
