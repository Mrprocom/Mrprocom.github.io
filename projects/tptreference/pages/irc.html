Internet Relay Chat (IRC)

QwebIRC

How to Connect

Enter a nickname you want to be seen as in the nickname text field, it is recommended that you pick a nickname that can be recognised by others as you. In the channels text field, the official IRC channel is already entered there, so you do not have to change it. If you do not have a NickServ account, ignore the Auth to services checkbox, if you have, check it and enter your username and password. Finally, complete the reCAPTCHA to confirm that you are not a robot and press Connect. If you want to learn more about IRC, check out the IRC Tutorial made by Mrprocom.

Brief Channel Description

Although this channel was intended to be for Powder Toy related discussions, you can talk about anything there as long as it is not violating the rules, users rarely talk about Powder Toy there. Do not expect the channel to be active all the time, users only talk when there is an interesting topic to talk about. The userlist on the right contains a list of users currently joined to the channel, but that does not mean they are currently online, they might be busy doing something else.

PowderBot is the only IRC bot there, it provides useful commands users can run to get automated answers, all of its commands start with a dollar sign $. When using PowderBot, make sure you do not spam the channel with it, if necessary, do your testings either in ##powder-bots or in a separate query with PowderBot using the /query command. The voiced users, who have a plus sign + at the beginning of their nicknames, moderate the channel, they can kick and ban other users who violate the rules.

When chatting in this channel, make sure you are following the rules and not being rude or annoying towards others, this way, you will be able to stay in the channel and not be hated by anyone.

Powder Toy Channels

ChannelDescription
#powder The main channel. Discussions about random topics, rarely about Powder Toy.
#powder-bots A bot testing channel. There are many IRC bots there, you can add your own if you have one. Almost all PowderBot commands are enabled there.
#powder-dev Discussions about the development of the game. Very inactive.
#powder-forum A channel where PowderBot automatically announces new forum threads and posts.
##powder-mc The main channel for the Powder Toy Minecraft server. Messages get relied between IRC and the Minecraft server.
#powder-moo A spam channel for running Crackbot commands.
#powder-social An off-topic channel. Users banned from the main channel can get redirected to this one.

Important PowderBot Commands

CommandDescriptionExample
$any <Nickname> Displays the last time that person did something (including joins and quits) and what their last action was. $any Ximon
$calc <Calculation> Solves the given calculation. $calc 8 * (5 + 8) / 4
$commands Returns a list of commands the bot has. $commands
$help <Command> Returns information on the given command $help seen
$justme <URL> Checks if that page is only down for you or for everyone. $justme google.com
$list [<Plugin>] Running the command with no arguments returns a list of plugins. Running the command with a plugin name returns a list of commands in that argument. $list powder
$ping Responds with a pong. Useful for checking if the bot is connected or to check if you are disconnected. $ping
$profile <Username> Returns information on the given Powder Toy account. $profile Simon
$seen <Nickname> Displays the last time that person talked and what their last message was. $seen Ximon
$wolf <Query> Sends the query to WolframAlpha and returns the output from there. $wolf 12 mile to feet
