Shortcuts

Notes:
  • LMB means left mouse button click or left click.
  • RMB means right mouse button click or right click.
  • MMB means middle mouse button click or middle click.
  • MB means mouse button click (left, right or middle).
  • + means and.
  • / means or.
  • - means to.

Mouse Shortcuts

ShortcutDescription
LMB Draws with the left-click element, which is the one highlighted with a red border. It can be changed by left-clicking a different element. By default, it is the DUST element.
RMB Draws with the left-click element, which is the one highlighted with a blue border. It can be changed by right-clicking a different element. By default, it is the erase particles element.
MMB Draws with the middle-click element, which is the one highlighted with a green border. It can be changed by middle-clicking a different element. By default, it is the sample tool.
Scroll Changes the brush size. It can also change the size of the zoom boxes before they are placed (hold Z).
Ctrl+Scroll Changes the vertical brush size.
Shift+Scroll Changes the horizontal brush size.

Single Key Shortcuts

ShortcutDescription
F1 Shows/hides help text, which is the same text you see when you first launch the game.
F2 Takes a screenshot and saves it in the main powder toy directory as a PNG image.
F3 Toggles debug mode.
F5 Reloads the current save.
0 - 9 Sets the view mode.
Q / Esc Quits the game.
W Changes the gravity mode (Vertical, Radial and Off).
E Opens up the element search dialog box.
R Records all frames and saves them in the main directory as PPM files. These can be then merged to create an animation. Press it again to stop recording.
Y Changes the air mode (On, Pressure Off, Velocity Off, Off and No Update).
U Toggles Ambient Heat mode.
I Inverts pressure and velocity.
P Takes a screenshot and saves it in the main powder toy directory as a PNG image.
[ Decreases the brush size. This can also decrease the size of zoom boxes before they are placed (hold Z).
] Increases the brush size. This can also increase the size of zoom boxes before they are placed (hold Z).
S Allows you to save stamps. After pressing S select an area to save it as a stamp, or right-click/press Esc to cancel. When there is STK2 in the simulation, press Ctrl+S to save stamps instead.
D Toggles debug mode. When there is STK2 in the simulation, press Shift+D to toggle debug mode instead.
F Go forward 1 frame (unpause the simulation for 1 frame and then pause it again).
G Shows grids. The grid size increases with each repetitive press, until grids become hidden.
H Toggles the heads up display (HUD).
K Opens up the stamps library.
L Loads the last saved or placed stamp.
Z Shows a zoom box for the area around the cursor. Releasing Z hides the zoom box. You can change the size of the zoom box using the scroll wheel or [ and ]. Click anywhere while holding Z to place the zoom box permanently on that location. Press Z again to remove the placed zoom box.
B Toggles decoration mode.
N Toggles sand effect mode.
~ Opens up the console.
= Resets pressure and velocity.
TAB Changes the shape of the brush (Circle, Square and Triangle).
Space Pauses/resumes the simulation.
Insert / ; Toggles Replace mode. The element that gets replaced is the one with a cyan border, which can be specified by pressing Ctrl+Alt+LMB on an element. The replacement element is the one with a red border, which can be specified by left-clicking an element.
Delete Toggles Specific Element Erase mode. The element that gets deleted is the one with a cyan border, which can be specified by pressing Ctrl+Alt+LMB on an element.

Multi-Key Shortcuts

ShortcutDescription
Ctrl+I Installs Powder Toy.
Ctrl+[ Decreases the brush size vertically.
Ctrl+] Increases the brush size vertically.
Ctrl+= Resets electricity.
Ctrl+F Toggles find mode. This will highlight the selected element from the simulation in red.
Ctrl+G Toggles display of newtonian gravity.
Ctrl+H Shows/hides help text, which is the same text you see when you launch the game.
Ctrl+; Toggles Specific Element Erase mode. The element that gets deleted is the one with a cyan border, which can be specified by pressing Ctrl+Alt+LMB on an element.
Ctrl+Z Undo.
Ctrl+X Allows you to cut an area and copy it. After pressing it, select an area to copy it or right-click/press Esc to cancel.
Ctrl+C Allows you to copy an area. After pressing it, select an area to copy it or right-click/press Esc to cancel.
Ctrl+V Allows you to paste a copied area. After pressing it, move your cursor to where you want to place it and left click to place it or right-click/press Esc to cancel.
Ctrl+B Toggles display of decoration layer.
Ctrl+Drag Draws rectangles.
Ctrl+Alt+Drag Draws squares.
Shift+1 Changes view mode to Life Gradient Display. This only works in debug mode.
Shift+[ Decreases the brush size horizontally.
Shift+] Increases the brush size horizontally.
Shift+G Shows grids. The grid size decreases with each repetitive press, until grids become hidden. This is the opposite of what pressing G does.
Shift+Drag Draws lines.
Shift+Alt+Drag Draws lines and snaps them in steps of 45 degrees.
Ctrl+Shift+MB Fills an enclosed area with the selected element. It depends on the mouse click, if it is a left-click, it will fill with the left-click element, if it is a right-click, it will fill with the right-click element and if it is a middle-click, it will fill with the middle-click element.
Ctrl+Shift+LMB Doing this on an element from the element selector adds it to the favourites category.
Ctrl+Shift+RMB Doing this on an element from the element selector removes it from the favourites category.
Alt+MB Samples the element the cursor is currently hovering over.

Shortcuts When Pasting

ShortcutDescription
R Rotates the selection counter-clockwise.
Shift+R Mirrors the selection horizontally.
Ctrl+Shift+R Mirrors the selection vertically.
Arrow Keys Moves the selection by one pixel in the direction of the pressed arrow key.
Esc / RMB Cancels the operation.

Stickmen Controls

ShortcutDescription
Left Arrow Makes STKM move to the left.
Right Arrow Makes STKM move to the right.
Up Arrow Makes STKM jump or fly if equipped with rocket boots.
Down Arrow Makes STKM shoot its Ctype particle.
A Makes STK2 move to the left.
D Makes STK2 move to the right.
W Makes STK2 jump or fly if equipped with rocket boots.
S Makes STK2 shoot its Ctype particle.
