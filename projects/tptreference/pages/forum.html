Powder Toy Forum

The Powder Toy Forum contains different categories you can make new threads in. If you want to make a new thread, make sure you make it in the proper category that matches your thread. Making new threads or posts requires signing in.

On the forum page, you will see three sections, one which lists the different categories, one which lists the important topics, which are pinned topics from the Announcements category and another one which lists the most recent threads. Pinned threads are the ones with a yellow icon, only moderators can pin threads. Locked threads are the ones with a lock icon, you cannot post in those threads, only moderators can lock and unlock threads. Threads can also be removed by moderators and moved to other categories if needed.

If you post in a thread, you will not be able to post again for 24 hours or until someone else posts. If have already posted, you can edit your post if you want.

Embedding Saves on the Forum

If you want to embed a save to your post, you can do so by typing your save ID after a tilde sign ~. For example, if you want to embed the Destroyable City 5 save, type ~2198 in your post.

Formatting Posts

Powder Toy posts are formatted using HTML tags, but not all HTML tags can be used, some tags like the script, style and iframe cannot be used. By default, accounts have WYSIWYG editor enabled, which stands for "What You See is What You Get", it is what you see when making a new post, it allows you to make text bold, italic, underlined, make links, post images and more. Sadly, you cannot use this editor in group posts, so you will have to use HTML tags to format the posts you make in Powder Toy groups. If you want, you can disable this editor by unchecking the last checkbox from your profile settings page.

If you decide to not use the WYSIWYG editor, you will need to know some HTML tags to help you format your posts. This is how HTML tags usually look like:

<tag>Visible Text</tag>

Some tags require attributes, which are additional information the tag uses, this is how they are added:

<tag attribute="value">Visible Text</tag>

For tags that do not show any text, this is how they look like:

<tag attribute="value">

HTML Entities

Some special characters (like letters from some languages) cannot be used in posts, instead, they automatically get replaced with question marks ?. To solve this problem, you have to use HTML entities of those special characters. HTML entities can be in three different forms:

  • &NAME; Example: &ohm; for the omega sign (Ω).
  • &#DEC; Example: &#937; for the omega sign (Ω).
  • &#xHEX; Example: &#x3A9; for the omega sign (Ω).

The first type is case-sensitive, for example, &sigma; is the HTML entity for small letter sigma (σ) while &Sigma; is the HTML entity for capital letter sigma (Σ). You can use online tools to easy convert special characters to their HTML entities, an example of a good one is this HTML entity encoder/decoder.
