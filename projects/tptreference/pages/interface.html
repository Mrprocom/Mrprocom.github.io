Game Interface

Heads Up Display (HUD)

  1. Framerate: FPS stands for Frames Per Second. It shows the number of frames the game is displaying in one second. The maximum is 60 FPS unless specified otherwise.
  2. Particle Count: The number of particles (except WALLs) currently on the simulation. It is only visible when debug mode is enabled, which can be toggled by pressing the D key.
  3. Mode: The mode that is currently being used. It is only visible when there is at least one mode activated. Available modes: Specific Delete, Replace Mode, and Find.
  4. Particle Type: The type of the particle the cursor is currently hovering over.
  5. Ctype: The secondary type (ctype) property of the particle the cursor is currently hovering over. It is only visible when debug mode is enabled, which can be toggled by pressing the D key.
  6. Temperature: The temperature of the particle the cursor is currently hovering over in Celsius.
  7. Life: The Life property of the particle the cursor is currently hovering over. This property slightly modifies the behavior of certain elements when modified (check the wiki page for any element to see whether changing the Life value has an effect on the element). It is only visible when debug mode is enabled, which can be toggled by pressing the D key.
  8. Tmp: The Tmp property of the particle the cursor is currently hovering over. This property slightly modifies the behavior of certain elements when modified (check the wiki page for any element to see whether changing the Tmp value has an effect on the element). It is only visible when debug mode is enabled, which can be toggled by pressing the D key.
  9. Pressure: The pressure of the spot under the cursor.
  10. Particle Index: The index or the ID of the particle (not the ID of the type of particle) the cursor is currently hovering over. It is only visible when debug mode is enabled, which can be toggled by pressing the D key.
  11. X Coordinate: The X (horizontal) coordinate of the cursor. It is only visible when debug mode is enabled, which can be toggled by pressing the D key.
  12. Y Coordinate: The Y (vertical) coordinate of the cursor. It is only visible when debug mode is enabled, which can be toggled by pressing the D key.
  13. Ambient Heat Temperature: The temperature of the air the cursor is currently hovering over. It is only visible when both the debug mode (toggled with the D key) and the ambient heat quick option (toggled with the U key) are enabled.

Quick Options and Categories

  1. Sand Effect: When enabled, it makes sand look fancier.
  2. Draw Gravity Field: When enabled, the game displays newtonian gravity fields (gravity fields generated by certain elements like BHOL and WHOL). Note that no gravity fields will be displayed if the newtonian gravity quick option is not enabled.
  3. Draw Decoration: When enabled, the game displays the decoration layer.
  4. Newtonian Gravity: When enabled, the game allows gravity generated by certain elements (like BHOL and WHOL) or certain tools (like PGRV and NGRV) to function.
  5. Ambient Heat: Enables ambient heat mode, which allows heat to travel through air.
  6. Show Console: Opens up the console, which allows you to run commands or Lua scripts that can perform various tasks in the game.
  7. Categories: Element categories contain a list of related elements.
  8. Favourites: A category for your bookmarked elements. You can bookmark an element by holding Ctrl + Shift while left clicking on the element. This can be reversed by holding Ctrl + Shift while right clicking on the element.
  9. Decoration Tools: A category of tools used for painting particles with any colour.
  10. Search: Opens up a search menu to search for any element, including hidden elements not listed under any category.

Toolbar

  1. Pause/Resume: Pauses/resumes the simulation.
  2. Render Options: Opens up a menu with render options where you can change the view mode or customise your own.
  3. Edit Profile/Sign In: Signs into the server with a Powder Toy account. Register a new account from the official Powder Toy website if you do not have one. If signed in, this button becomes an Edit Profile button with some account settings.
  4. Sign In: If not signed in, this button becomes part of the previously mentioned button. If signed in, the new button will open up a dialog box for signing out or signing into a different account.
  5. Clear Simulation: Resets the simulation.
  6. Tags: Shows a list of tags placed on the current save. Click it to add tags to the current save (requires being signed in). Tags are used to help improve search results.
  7. Vote Down: Votes down the current save (requires being signed in).
  8. Vote Up: Votes up the current save (requires being signed in).
  9. Save: Uploads your simulation to the server if you are signed in. If not, it will save your simulation locally so that you can access them offline. You still save locally while being signed in by holding Ctrl while clicking it.
  10. Re-upload: Updates the current simulation.
  11. Reload: Reloads the current simulation. Right clicking the reload button will show information about the current save.
  12. Browse: Opens up the save browser where you can find saves made by the community. You can also hold Ctrl while right clicking the browse button to browse through your local saves.

Element Selector

  1. Left-click Element: The element that gets placed when drawing with left-click. It can be changed by left-clicking a different element. By default, it is the DUST element.
  2. Right-click Element: The element that gets placed when drawing with right-click. It can be changed by right-clicking a different element. By default, it is the erase particles tool.
  3. Middle-click Element: The element that gets placed when drawing with middle-mouse button. It can be changed by middle-clicking a different element. By default, it is the sample tool.
  4. Selected Element to be Replaced: The element that is used to be replaced in Specific Replace mode, or deleted in Specific Delete mode. It can be changed by holding Ctrl+Alt while clicking an element. By default, it is the erase particles element.
  5. Elements: Unselected elements that can be selected to draw with.
