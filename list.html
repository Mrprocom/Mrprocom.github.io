Homepage

This project is the best one yet. It took me over 2 months to make it.

This web page is supposed to be used as a homepage. It contains about 200 useful, fun and popular websites arranged into sections. It is also very customisable, you can add an unlimited number of pages to it if you want.

This project went through many upgrades, the current version uses the metro theme inspired by Windows 8 start screen. Over 60% of all icons used were taken from Flaticon.

Visit Homepage

IRC Quote Prettifier

This project is a useful tool used to prettify IRC quotes.

It has options to change the message syntaxes, hide IPs and hostmasks, colour nicknames, convert IRC colours and text formats to HTML, HTML encode special characters and preview the end result.

This tool is useful for posting IRC quotes into websites that support HTML formats.

Visit IRC Quote Prettifier

IRC Tutorial

This project is a tutorial on how to use IRC properly.

The main intention to making this is to give the URL of the tutorial to whoever is encountering problems with IRC. It is also useful for telling people who are constantly acting inappropriately how to behave properly.

It is not that fancy, it just has Webchat embedded to it with buttons under it that list different information about different topics.

Visit IRC Tutorial

This project is a tutorial on how to organise Powder Toy posts using HTML tags. It is also useful for getting started with HTML.
